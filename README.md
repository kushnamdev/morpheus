# Morpheus

Morpheus is an AI video creator that can convert your articles to a video with just URL

## Installation



```bash
cd frontend
npm install

cd backend
npm install
```

## Usage

```bash
#to run frontend server
cd frontend
npm run dev

#to run backend server
cd backend
node index.js
```
